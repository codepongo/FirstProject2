Detailed Description

The Object class represents the root class for most classes, and provides some basic Type and conversion services. This class has no datamembers and only adds one vtable entry as overhead. This class is by default allocated in the garbage collected heap and do not need an explicit delete. The current implementation however allows delete and frees the object. Using the object after deleting results in undefined behavior.
